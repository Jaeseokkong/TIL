# 💼 격자판 저수지 찾기 문제

## 🧾 문제 설명
N×N 격자판에 각 지역의 **물 깊이**가 주어집니다.
자신의 상, 하, 좌, 우의 깊이보다 더 깊은 지역은 저수지로 분류합니다.
격자의 가장자리는 깊이가 0이라고 가정합니다.

### 📥 입력 예시
```js
let depthMap = [
  [4, 2, 5, 1, 3],
  [3, 6, 2, 7, 2],
  [5, 3, 8, 4, 6],
  [4, 5, 7, 3, 2],
  [6, 4, 2, 5, 1]
];
```
- 각 숫자는 해당 지역의 물 깊이 (정수, 최대 100)

### 🎯 목표
격자판에서 저수지의 개수를 구하시오.
