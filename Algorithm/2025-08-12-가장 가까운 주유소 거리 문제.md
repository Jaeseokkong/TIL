# 💼 가장 가까운 주유소 거리 문제

## 문제 설명
고속도로를 나타내는 문자열 `road` 가 주어집니다.
문자열에는 빈 도로(`.`)와 주요소(`G`)가 표시되어 있습니다.
각 도로 위치에서 가장 가까운 주유소(`G`)까지의 **최소 거리**를 계산하십시오.

### 📥 입력 예시
```js
let road = "..G....G..G.";
```

### 📥 출력 예시
```js
[2, 1, 0, 1, 2, 3, 0, 1, 0, 1, 2, 3]
```

### 목표
- 각 인덱스의 문자에서 `G` 까지 떨어진 최소 거리를 구하기
- 도로 길이는 최대 100, `G` 는 하나 이상 존재
